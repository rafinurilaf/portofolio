##  Cara Jalankan

1. Clone repository ini:
```bash
git clone https://github.com/rafinurilaf/portofolio.git
cd portofolio
````

2. Install dependencies:
```bash
npm install
````

3. Jalankan server lokal dengan hot reload:
```bash
npm run dev
````

4. Buat versi production:
```bash
npm run build
````
